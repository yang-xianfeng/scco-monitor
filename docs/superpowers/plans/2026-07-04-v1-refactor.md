# SCCO Monitor v1.0 Refactor — Implementation Plan

> **For agentic workers:** Inline execution in current session.

**Goal:** Refactor SCCO Monitor into a type-safe, self-documenting 1.0 codebase.

**Architecture:** Add `models.py` for TypedDict data models + Signal enum; rename `backtest.py` → `zone.py`; type-annotate all module boundaries; delete `_row_to_numeric` from main.py.

**Tech stack:** Python 3.12, yfinance, pandas, requests, pytest

## Global Constraints

- All existing tests must pass unchanged
- No functional changes: ratio calc, threshold logic, non-trading-day fallback all identical
- Template HTML, GitHub Actions, CSV format untouched
- TypedDict is pure type annotation — zero runtime overhead
