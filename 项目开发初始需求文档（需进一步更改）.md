# 📋 项目开发文档：Cola-SCCO 辅助交易与通知系统 (简称: Cola-Link)

## 1. 项目概述

### 1.1 背景与目的

本系统旨在将 Cola 哥对 SCCO 的“铜价系数模型”及仓位管理策略进行自动化落地。系统通过实时/定时获取铜期货价格与 SCCO 股票数据，动态计算“当前杠杆率系数”，并根据预设区间提供交易信号提示、弹性的仓位风险预警，辅助投资者做出理性决策。

### 1.2 核心业务逻辑

1. **输入数据**：铜现货/期货价格（$P_{copper}$）、SCCO 当前股价（$P_{scco}$）、SCCO 总股本（$Shares$）。
2. **核心公式**：
* $\text{公式市值 (Anchor Market Cap)} = \frac{P_{copper}}{4.2} \times 900 亿$
* $\text{实际市值 (Actual Market Cap)} = P_{scco} \times Shares$
* $\text{当前系数 (Leverage Ratio)} = \frac{\text{实际市值}}{\text{公式市值}}$


3. **区间决策矩阵**：

| 系数区间 | 市场状态 | 对应 SCCO 价格区间 (以 $6.29 铜价为例) | 核心操作信号 | 组合层调整建议 |
| --- | --- | --- | --- | --- |
| **$\le 1.10$** | 偏安全 / 可关注 | $\le \$177.84$ | 🟢 逢低分批买入/持有 | 允许维持高 Beta |
| **$1.10 \sim 1.20$** | 合理区间 | $\$177.84 \sim \$194.00$ | 🟡 持有，不盲目追高 | 保持观望 |
| **$1.20 \sim 1.50$** | 偏热，弹性释放 | $\$194.00 \sim \$242.51$ | 🟠 停止买入，停止Long Call | 提示资源仓位整体降 Beta |
| **$\ge 1.50$** | 极值 / 减仓区 | $\ge \$242.51$ | 🔴 分批减仓 / 卖出末日 Short Call | 强制降低资源股总仓位 |

## 2. 系统架构与技术选型（追求“简单高效”）

为了避免复杂的服务器维护和高昂的行情 API 费用，采用轻量化、无服务器（Serverless）架构。

```
[数据源: Yahoo Finance API / 爬虫] 
              │ (每30分钟/闭市后触发)
              ▼
    [GitHub Actions / AWS Lambda] ──(运行 Python 核心计算脚本)
              │
              ▼ [触发条件判断: 跨区间 / 到达目标价]
    [通知推送: 微信(Server酱)/Telegram/飞书机器人]

```

* **开发语言**：Python 3.10+（金融计算与数据处理首选）
* **数据源（免费且稳定）**：`yfinance` 库（获取 SCCO 股价、总股本及铜期货 `HG=F` 的数据）
* **运行环境**：GitHub Actions（完全免费，支持定时任务 `cron`，无需服务器）
* **通知渠道**：飞书机器人 / Webhook / Telegram Bot / Server酱（微信推送）

## 3. 功能模块设计

### 模块 A：数据采集与清洗 (Data Ingestion)

* **任务**：获取当前最新的铜价和 SCCO 股价。
* **逻辑**：
* SCCO 股本数可在脚本中设为固定常数（约 7.73 亿股，定期更新即可），或通过 API 动态获取。
* 获取纽约商品交易所（COMEX）高级铜期货（代码：`HG=F`）的最新价格。



### 模块 B：核心计算引擎 (Calculation Engine)

* **任务**：运行 Cola 系数公式。
* **输出**：当前真实系数、当前系数对应的各区间绝对价格边界、当前用户的持仓盈亏及风险度。

### 模块 C：信号触发与通知模块 (Alert & Notification)

* **触发机制**：
1. **定时汇报**：每天美股开盘前、收盘后各推送一次日振幅快报。
2. **阈值触发**：当系数跨越重要关口（如跌破 1.10，或突破 1.50）时，触发高优先级警报。
3. **风控提示**：当检测到用户处于浮亏状态（如当前 -4.6%），且系数 $\le 1.10$ 时，提示：“估值安全，无需恐慌砍仓；若组合总 Beta 过高，请从其他高 Beta 标的减仓，而非 SCCO”。

## 4. 核心代码实现 (Python 极简原型)

```python
import yfinance as yf
import requests

# ==================== 配置区域 ====================
# 用户持仓配置（用于个性化风控提示）
USER_COST = 184.36
FEISHU_WEBHOOK = "https://open.feishu.cn/open-apis/bot/v2/hook/xxxx"

def get_market_data():
    # 获取铜期货和SCCO最新价格
    copper = yf.Ticker("HG=F")
    scco = yf.Ticker("SCCO")
    
    # 确保获取到的是最新价格
    p_copper = copper.history(period="1d")['Close'].iloc[-1]
    p_scco = scco.history(period="1d")['Close'].iloc[-1]
    
    # SCCO总股本 (可动态获取或写死，当前约 7.73亿)
    shares = scco.info.get('sharesOutstanding', 773000000)
    
    return p_copper, p_scco, shares

def calculate_cola_model(p_copper, p_scco, shares):
    actual_market_cap = (p_scco * shares) / 1e9  # 换算成亿美元
    
    # Cola 核心公式: 4.2 铜价对应 900亿市值
    anchor_market_cap = (p_copper / 4.2) * 900
    
    # 当前杠杆率系数
    ratio = actual_market_cap / anchor_market_cap
    
    # 反推各区间SCCO价格
    scco_p_110 = (1.10 * anchor_market_cap * 1e9) / shares
    scco_p_120 = (1.20 * anchor_market_cap * 1e9) / shares
    scco_p_150 = (1.50 * anchor_market_cap * 1e9) / shares
    
    return ratio, anchor_market_cap, actual_market_cap, scco_p_110, scco_p_120, scco_p_150

def push_notification(text):
    # 示例：飞书机器人推送
    payload = {"msg_type": "text", "content": {"text": text}}
    requests.post(FEISHU_WEBHOOK, json=payload)

def main():
    p_copper, p_scco, shares = get_market_data()
    ratio, anchor_cap, actual_cap, p_110, p_120, p_150 = calculate_cola_model(p_copper, p_scco, shares)
    
    # 判断信号
    if ratio <= 1.10:
        signal = "🟢 偏安全/可关注（当前并非估值过热，跌回安全边际）"
        action_advice = "不建议因估值砍仓。若需降低组合Beta，应从其他标的下手。"
    elif ratio <= 1.20:
        signal = "🟡 合理区间（持有，不追）"
        action_advice = "保持观望，不建议追高。"
    elif ratio < 1.50:
        signal = "🟠 偏热（弹性释放中）"
        action_advice = "停止加仓，逐步停止Long Call。"
    else:
        signal = "🚨 减仓区（强烈警惕）"
        action_advice = "严格执行减仓原则，可配合分批减仓或布防末日Short Call。"
        
    user_profit_loss = ((p_scco - USER_COST) / USER_COST) * 100

    # 组装推送文本
    report = f"""
【Cola-SCCO 辅助交易报告】
────────────────
📊 实时行情:
• 当前铜价: ${p_copper:.4f}/lb
• SCCO 股价: ${p_scco:.2e}
• 您的持仓成本: ${USER_COST} (当前浮亏: {user_profit_loss:.2f}%)

📐 模型计算:
• 模型合理市值: ${anchor_cap:.2f} 亿
• SCCO 实际市值: ${actual_cap:.2f} 亿
• 📊 当前系数: {ratio:.4f}

🛑 价格区间参考 (基于当前铜价):
• 1.10 安全上沿: ${p_110:.2f}
• 1.20 合理上沿: ${p_120:.2f}
• 1.50 减仓起点: ${p_150:.2f}

📢 决策信号: {signal}
💡 组合操作建议: {action_advice}
────────────────
"""
    print(report)
    # push_notification(report) # 取消注释开启推送

if __name__ == "__main__":
    main()

```

## 6. 后续扩展规划 (辅助交易进阶)

1. **多标的矩阵**：可将此模型横向扩展至其他铜矿巨头（如 FCX、TECK），对比它们之间的杠杆率偏离度，做**同行配对交易 (Pair Trading)**。
2. **期权隐含波动率（IV）监控**：当系数 $\ge 1.50$ 且 SCCO 的 IV 处于历史高位时，系统自动高亮提示：“进入末日 Short Call 的绝佳统计学区间”。
3. **仓位自动扣减建议**：在通知中直接根据 60%/30%/10% 的原则，计算出当前铜价下，正股和期权各自应调整到的目标股数/张数。


可参考的 案例源码：
#!/usr/bin/env python3
# -*- coding: utf-8 -*-
"""
铜期货与SCCO股价实时监控脚本
使用Yahoo Finance API获取实时数据，并计算SCCO相对铜价的溢价率

溢价率公式：(SCCO股价 × 总股本) × 4.2 / 900亿 / 铜期货价格
"""

from __future__ import annotations

import time
import os
from datetime import datetime

import yfinance as yf

# 配置
COPPER_TICKER = "HG=F"  # 纽约铜期货
SCCO_TICKER = "SCCO"    # 南方铜业
UPDATE_INTERVAL = 60    # 更新间隔（秒）

# 溢价率计算常数
# 4.2：经验倍数，反映SCCO铜资源的开采年限折算因子
PREMIUM_MULTIPLIER = 4.2
# 900亿：参考基准值（单位：元），代表行业内铜资源的基准估值锚点（约900亿元）
PREMIUM_DIVISOR = 900 * 1e8     # 900亿 = 9e10


class CopperMarketMonitor:
    """铜市场实时监控器"""

    def fetch_info(self, ticker: str) -> dict:
        """通过 Yahoo Finance 获取股票/期货的实时信息"""
        try:
            t = yf.Ticker(ticker)
            info = t.info
            return info
        except Exception as e:
            print(f"获取 {ticker} 数据失败: {e}")
            return {}

    def display_copper_futures(self, info: dict) -> float | None:
        """显示铜期货数据，返回当前价格（美元/磅）"""
        if not info:
            print("❌ 无法获取铜期货数据")
            return None

        current = info.get("regularMarketPrice")
        change_pct = info.get("regularMarketChangePercent", 0)
        previous = info.get("previousClose")
        day_high = info.get("dayHigh")
        day_low = info.get("dayLow")
        week52_high = info.get("fiftyTwoWeekHigh")
        week52_low = info.get("fiftyTwoWeekLow")

        change_color = "🟢" if change_pct >= 0 else "🔴"

        print("\n" + "=" * 60)
        print("🟤 纽约铜期货 (HG=F) - COMEX")
        print("=" * 60)
        print(f"当前价格:    ${current:.4f} /磅")
        print(f"涨跌幅度:    {change_color} {change_pct:+.2f}%")
        if previous is not None:
            print(f"昨收价格:    ${previous:.4f}")
        if day_high is not None:
            print(f"今日最高:    ${day_high:.4f}")
        if day_low is not None:
            print(f"今日最低:    ${day_low:.4f}")
        if week52_high is not None:
            print(f"52周最高:    ${week52_high:.2f}")
        if week52_low is not None:
            print(f"52周最低:    ${week52_low:.2f}")

        return current

    def display_scco_stock(self, info: dict) -> tuple[float | None, float | None]:
        """显示SCCO股票数据，返回 (当前股价, 总股本)"""
        if not info:
            print("❌ 无法获取SCCO数据")
            return None, None

        current = info.get("regularMarketPrice")
        change_pct = info.get("regularMarketChangePercent", 0)
        previous = info.get("previousClose")
        day_high = info.get("dayHigh")
        day_low = info.get("dayLow")
        market_cap = info.get("marketCap")
        trailing_pe = info.get("trailingPE")
        dividend_yield = info.get("dividendYield")
        beta = info.get("beta")
        shares_outstanding = info.get("sharesOutstanding")

        change_color = "🟢" if change_pct >= 0 else "🔴"

        print("\n" + "=" * 60)
        print("🏭 南方铜业公司 (SCCO) - NYSE")
        print("=" * 60)
        print(f"当前股价:    ${current:.2f}")
        print(f"涨跌幅度:    {change_color} {change_pct:+.2f}%")
        if previous is not None:
            print(f"昨收价格:    ${previous:.2f}")
        if day_high is not None:
            print(f"今日最高:    ${day_high:.2f}")
        if day_low is not None:
            print(f"今日最低:    ${day_low:.2f}")
        if market_cap is not None:
            print(f"市值:        ${market_cap / 1e9:.2f}B")
        if trailing_pe is not None:
            print(f"市盈率:      {trailing_pe:.2f}")
        if dividend_yield is not None:
            print(f"股息率:      {dividend_yield * 100:.2f}%")
        if beta is not None:
            print(f"Beta系数:    {beta:.3f}")
        if shares_outstanding is not None:
            print(f"总股本:      {shares_outstanding:,.0f} 股")

        return current, shares_outstanding

    def calculate_premium(
        self,
        scco_price: float,
        shares_outstanding: float,
        copper_price: float,
    ) -> float:
        """
        计算SCCO相对铜价的溢价率

        公式：(SCCO股价 × 总股本) × 4.2 / 900亿 / 铜期货价格
        """
        market_cap = scco_price * shares_outstanding
        premium_ratio = market_cap * PREMIUM_MULTIPLIER / PREMIUM_DIVISOR / copper_price
        return premium_ratio

    def display_premium(
        self,
        scco_price: float | None,
        shares_outstanding: float | None,
        copper_price: float | None,
    ) -> None:
        """显示SCCO相对铜价的溢价率"""
        if None in (scco_price, shares_outstanding, copper_price):
            return

        premium_ratio = self.calculate_premium(scco_price, shares_outstanding, copper_price)

        print("\n" + "=" * 60)
        print("📊 SCCO 相对铜价溢价率")
        print("=" * 60)
        print(f"  公式: (SCCO股价 × 总股本) × {PREMIUM_MULTIPLIER} / 900亿 / 铜期货价格")
        print(f"  SCCO股价:     ${scco_price:.2f}")
        print(f"  总股本:       {shares_outstanding:,.0f} 股")
        print(f"  铜期货价格:   ${copper_price:.4f}/磅")
        market_cap = scco_price * shares_outstanding
        print(f"  SCCO市值:     ${market_cap / 1e9:.2f}B")
        print(f"  溢价率:       {premium_ratio:.4f}  ({premium_ratio * 100:.2f}%)")

    def run_once(self) -> None:
        """执行单次数据获取和显示"""
        # 获取数据
        copper_info = self.fetch_info(COPPER_TICKER)
        scco_info = self.fetch_info(SCCO_TICKER)

        # 清屏
        os.system("clear" if os.name != "nt" else "cls")

        # 显示标题
        print("\n" + "█" * 60)
        print("█" + " " * 20 + "铜市场实时监控" + " " * 20 + "█")
        print("█" * 60)
        print(f"更新时间: {datetime.now().strftime('%Y-%m-%d %H:%M:%S')}")

        # 显示各板块数据并收集计算所需数值
        copper_price = self.display_copper_futures(copper_info)
        scco_price, shares_outstanding = self.display_scco_stock(scco_info)

        # 显示溢价率
        self.display_premium(scco_price, shares_outstanding, copper_price)

        print("\n" + "=" * 60)
        print(f"⏱️  下次更新: {UPDATE_INTERVAL}秒后 | 按Ctrl+C退出")
        print("=" * 60)

    def run_continuous(self) -> None:
        """持续监控模式"""
        print("\n启动实时监控模式...")
        print(f"更新间隔: {UPDATE_INTERVAL}秒")
        print("按 Ctrl+C 停止监控\n")

        try:
            while True:
                self.run_once()
                time.sleep(UPDATE_INTERVAL)
        except KeyboardInterrupt:
            print("\n\n监控已停止")


def main() -> None:
    monitor = CopperMarketMonitor()

    # 单次运行（用于测试）
    print("\n单次数据获取模式:")
    monitor.run_once()

    # 持续监控模式（取消注释以启用）
    # monitor.run_continuous()


if __name__ == "__main__":
    main()
